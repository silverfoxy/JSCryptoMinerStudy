<p style="font-weight: bold; font-family: sans-serif;">Welcome to ntp.org, home of the Network Time Protocol project.</p>

<p>This page is home for the NTP Project (R&amp;D).
This web site, and all pages directly contained within, are in the
public domain. Some portions of this site may be copyrighted by other authors.
Please contact the webmaster with any questions regarding copyright. The NTP
software distribution is copyrighted, as described in the <a
href="http://www.eecis.udel.edu/~mills/ntp/html/copyright.html">NTP copyright page</a>.</p>

<p style="font-weight: bold; font-family: sans-serif;">NTP is a protocol designed to synchronize the clocks of computers over a network.</p>

<p>NTP version 4, a significant revision of the previous NTP standard, is the current development version. It is formalized by <a href="rfc.html">RFCs</a> released by the IETF.</p>

<p style="font-weight: bold; font-family: sans-serif;">The NTP Project (R&amp;D)</p>

<p>The NTP Project conducts Research and Development in NTP and produces the
Official Reference Implementation of NTP along with the <a
href="http://www.eecis.udel.edu/~mills/ntp/html/index.html">Implementation
Documentation</a>.  Background information about NTP, along with briefings and
a bibliography, are available at the <a
href="http://www.eecis.udel.edu/~mills/ntp.html">Network Time Synchronization
Project Page</a>.</p>
