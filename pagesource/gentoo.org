    <h2 class="stick-top newsitem-headline">
      <a href="/news/2018/02/19/Gentoo-GSoC-2018.html">Gentoo accepted into Google Summer of Code 2018</a> <small>(Feb 19, 2018)</small>
    </h2>
    <div class="newsitem-content">
      <p>Students who want to spend their summer having fun and writing code can do so now for Gentoo. Gentoo <a href="https://summerofcode.withgoogle.com/organizations/4918228900380672/">has been accepted</a> as a mentoring organization for this year’s Google Summer of Code.</p>

<p>The GSoC is an excellent opportunity for gaining real-world experience in software design and making one’s self known in the broader open source community. It also looks great on a resume.</p>

<p>Initial project ideas can be <a href="https://wiki.gentoo.org/wiki/Google_Summer_of_Code/2018/Ideas">found here</a>, although new projects ideas are welcome. For new projects time is of the essence: there is typically some idea-polishing which must occur before the <strong><em>March 27th deadline</em></strong>. Because of this it is strongly recommended that students refine new project ideas with a mentor <em>before</em> proposing the idea formally.</p>

<p>GSoC students are encouraged to begin discussing ideas in the <a href="https://webchat.freenode.net/?channels=gentoo-soc">#gentoo-soc</a> IRC channel on the Freenode network.</p>

<p>Further information can be found on the <a href="https://wiki.gentoo.org/wiki/Google_Summer_of_Code/2018">Gentoo GSoC 2018 wiki page</a>. Those with unanswered questions should not hesitate to <a href="mailto:soc-mentors@gentoo.org">contact</a> the Summer of Code mentors via the mailing list.</p>

      
    </div>
  </article>

    <h2 class="stick-top newsitem-bullet">

    <h2 class="stick-top newsitem-headline">
      <a href="/news/2018/01/30/teh-fosdem-is-near.html">FOSDEM is near</a> <small>(Jan 30, 2018)</small>
    </h2>
    <div class="newsitem-content">
      <p><!-- Yes, I almost went with "Teh FOSDEM nearth", but decided against it since 'nearth' is not a real word. Nevermind the 'Teh', mellenials love using 'teh'. Gentoo can be fun! =) --></p>

<p>Excitement is building with FOSDEM 2018 only a few days away. There are now <a href="https://wiki.gentoo.org/wiki/FOSDEM_2018#Developers">14 current</a> and <a href="https://wiki.gentoo.org/wiki/User:Dberkholz">one former</a> developer in planned attendance, along with many from the Gentoo community.</p>

<p>This year one Gentoo related talk titled <em><a href="https://fosdem.org/2018/schedule/event/unix_windows_gentoo/">Unix? Windows? Gentoo!</a></em> will be given by <a href="https://fosdem.org/2018/schedule/speaker/michael_haubenwallner/">Michael Haubenwallner</a> (<a href="https://wiki.gentoo.org/wiki/User:Haubi">haubi <i class="fa fa-user" aria-hidden="true"></i></a>) of the <a href="https://wiki.gentoo.org/wiki/Project:Prefix">Gentoo Prefix</a> project.</p>

<p>Two more non-Gentoo related talks will be given by current Gentoo developers:</p>

<ul>
  <li><a href="https://fosdem.org/2018/schedule/speaker/andreas_k_huettel/">Andreas K. Huettel</a> (<a href="https://wiki.gentoo.org/wiki/User:Dilfridge">dilfridge <i class="fa fa-user" aria-hidden="true"></i></a>) – <em><a href="https://fosdem.org/2018/schedule/event/perl_physics/">Perl in the Physics Lab</a></em></li>
  <li><a href="https://fosdem.org/2018/schedule/speaker/andrew_savchenko/">Andrew Savchenko</a> (<a href="https://wiki.gentoo.org/wiki/User:Bircoph">bircoph <i class="fa fa-user" aria-hidden="true"></i></a>) – <em><a href="https://fosdem.org/2018/schedule/event/i2p/">The Invisible Internet Project</a></em></li>
</ul>

<p>If you attend don’t miss out on your opportunity to build the Gentoo web-of-trust by bringing a valid governmental ID, a printed key fingerprint, and various UIDs you want certified. See you at FOSDEM!</p>

      
    </div>
