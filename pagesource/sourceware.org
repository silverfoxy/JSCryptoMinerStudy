<p>Occasionally asked questions:
</div>

<dl>

<dt><b>I can't access the ftp site.  Mirrors?</b></dt>
<dd>
   There are several sites which mirror the 
   <a href="ftp://sourceware.org/pub/">free software ftp site</a>.  You 
   can see the full list on the <a href="mirrors.html">free software mirrors 
   list page</a>.
   <p>
</dd>

<dt><b>How are the projects chosen?  
       Can I host my free software project there?</b></dt>
<dd>
    Currently, we are pleased to be able to provide a home for Red Hat's
    development tool offerings and certain
<a href="http://www.gnu.org/">GNU Project</a>
packages such as
<a href="http://gcc.gnu.org/">GCC</a>, <a href="gdb/">GDB</a>, <a href="glibc/">glibc</a>,
    <a href="binutils/">Binutils</a>, and others.  However, we currently only have the capacity to
    host a limited number of projects.
   <br>So, while it doesn't hurt to
   <a href="mailto:sourcemaster@sourceware.org" >ask</a> if your project
   can be hosted here, we do have to be selective in what we offer.
   <br>With luck, this will change soon, as we are contemplating increasing the capacity
   of both the sourceware.org system and the internet connection that it uses to
   communicate with the outside world.
   <p>
</dd>

<dt><b>I need an account on sourceware for git/svn/cvs write access to project <i>XYZ</i>?</b></dt>
<dd>
   We have an XYZ Project? Cool!

   <br>
     Ok, you need write access for a project hosted on sources you say? Well, first you'll need to have your request approved by the project maintainer(s). Once they give their ok, you can use our <a href="/cgi-bin/pdw/ps_form.cgi">handy dandy little form</a> and we'll process your request.  This will provide you with an account on sourceware.org.  If, in the future, you want to
contribute to the repository of another project (remember that gcc.gnu.org and sourceware.org share projects), have a responsible party send email to the overseers mailing list at
this site requesting that you be added to the project.  <em>Do not fill this form out again.</em>
   <p>
</dd>

<dt><b>Do you have a privacy statement for this site?</b></dt>
<dd>
   Hey, why not.
<br>
     We're not recording anything about you, short of the normal web logs
     which show what pages are being downloaded by what host.  We'd never
     use any of the mailing list subscription information for anything
     other than the normal distribution of e-mail to that list.  
<br><br>We are not out to get you.  

<br><br>
     Oh, from time to time we do generate some aggregate reports but those
     don't identify you individually, so stop worrying.

   <p>
</dd>

<dt><b>A lot of the software here is compressed in the .bz2 format.  
    What is that?</b></dt>
<dd>
   bzip2 ("<tt>.bz2</tt>") is a compression program similar to gzip
   ("<tt>.gz</tt>"), but it produces much smaller files than gzip.  You
   can get the source for bzip2, or pre-built binaries for most platforms,
   from the <a href="http://www.bzip.org/">bzip2 home page</a>.
   Most recent free software distributions include bzip2, so you may already
   have it on your system.
   <p>
</dd>

<dt><b>Is it true that the first lunar landing was faked?</b></dt>
<dd>
   Of course.  Everyone knows that.  See the PBS documentary
   about it, <a href="http://us.imdb.com/Title?Capricorn+One+(1978)">Capricorn One</a>.
  <p>
</dd>
