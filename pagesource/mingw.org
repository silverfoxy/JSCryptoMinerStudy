    <h3 id="toc0">Home of the MinGW and MSYS Projects</h3>
<p>
</p>
<hr />
<p>
<strong>MinGW</strong>, a contraction of &quot;Minimalist GNU for Windows&quot;, is a minimalist development environment for native Microsoft Windows applications.</p>
<p><strong>MinGW</strong> provides a complete Open Source programming tool set which is suitable for the development of native MS-Windows applications, and which do not depend on any 3rd-party C-Runtime DLLs.  (It <em><strong>does</strong></em> depend on a number of DLLs provided by Microsoft themselves, as components of the operating system; most notable among these is MSVCRT.DLL, the Microsoft C runtime library.  Additionally, threaded applications <em><strong>must</strong></em> ship with a freely distributable thread support DLL, provided as part of MinGW itself).</p>
<p><strong>MinGW</strong> compilers provide access to the functionality of the Microsoft C runtime and some language-specific runtimes. <strong>MinGW</strong>, being Minimalist, <strong>does not, and never will, attempt to provide a POSIX runtime environment for POSIX application deployment on MS-Windows</strong>. If you want POSIX application deployment on this platform, please consider <a href="http://www.cygwin.com">Cygwin</a> instead.</p>
<p>Primarily intended for use by developers working on the native MS-Windows platform, but also available for cross-hosted use, (see note below -- you may need to follow the &quot;<a href="/Welcome_to_MinGW_org">read more</a>&quot; link to see it), <strong>MinGW</strong> includes:</p>
<ul>
<li> A port of the GNU Compiler Collection (GCC), including C, C++, ADA and Fortran compilers;</li>
<li> GNU Binutils for Windows (assembler, linker, archive manager)</li>
<li> A command-line installer, with optional GUI front-end,  (mingw-get) for MinGW and MSYS deployment on MS-Windows</li>
<li> A GUI first-time setup tool (mingw-get-setup), to get you up and running with mingw-get.</li>
</ul>
<p><strong>MSYS</strong>, a contraction of &quot;Minimal SYStem&quot;, is a Bourne Shell command line interpreter system. Offered as an alternative to Microsoft's cmd.exe, this provides a general purpose command line environment, which is particularly suited to use with <strong>MinGW</strong>, for porting of many Open Source applications to the MS-Windows platform; a light-weight fork of Cygwin-1.3, it includes a small selection of Unix tools, chosen to facilitate that objective.</p>
