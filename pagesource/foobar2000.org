  <ul class="bulletlist">
    <li>Supported audio formats: MP3, MP4, AAC, CD Audio, WMA, Vorbis, Opus, FLAC, WavPack, WAV, AIFF, Musepack, Speex, AU, SND... and more with additional <a href="/?page=Download#components">components</a>.</li>
    <li><a href="http://wiki.hydrogenaudio.org/index.php?title=Gapless" rel="nofollow">Gapless</a> playback.</li>
    <li>Easily <a href="http://wiki.hydrogenaudio.org/index.php?title=Foobar2000:Layout_Editing_Mode" rel="nofollow">customizable user interface layout</a>.</li>
    <li>Advanced <a href="http://wiki.hydrogenaudio.org/index.php?title=Foobar2000redirect:C16D48EE-39BE-4C91-9A35-441BEFA286D2" rel="nofollow">tagging capabilities</a>.</li>
    <li>Support for ripping Audio CDs as well as transcoding all supported audio formats using the <a href="/FAQ#converting_audio_files_to_different_file_formats">Converter component</a>.</li>
    <li>Full <a href="http://wiki.hydrogenaudio.org/index.php?title=Replaygain" rel="nofollow">ReplayGain</a> support.</li>
    <li>Customizable keyboard shortcuts.</li>
    <li>Open component architecture allowing third-party developers to extend functionality of the player.</li>
