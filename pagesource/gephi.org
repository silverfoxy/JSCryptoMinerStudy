                <h2>Applications</h2>
                <p class="tick"><b>Exploratory Data Analysis</b>: intuition-oriented analysis by networks manipulations in real time.</p>
                <p class="tick"><b>Link Analysis</b>: revealing the underlying structures of associations between objects.</p>
                <p class="tick"><b>Social Network Analysis</b>: easy creation of social data connectors to map community organizations and small-world networks.</p>
                <p class="tick"><b>Biological Network analysis</b>: representing patterns of biological data.</p>
                <p class="tick"><b>Poster creation</b>: scientific work promotion with hi-quality printable maps.</p>
                <span class="learn_more"><a href="/features/">Learn More »</a></span>

                <hr />
                <h2>Metrics ready</h2>
                <p class="tick"><b>Centrality</b>: used in sociology to indicate how well a node is connected. Available: degree (power-law), betweenness, closeness.</p>
                <p class="tick"><b>And more</b>: density, path length, diameter, HITS, modularity, clustering coefficient.</p>
                <span class="learn_more"><a href="/features/">Learn More »</a></span>

                <hr />
                <h2>Technology</h2>
                <p class="tick"><b>Ergonomic interface</b>: no programming skills needed</p>
                <p class="tick"><b>High-performance</b>: built-in rendering engine.</p>
                <p class="tick"><b>Native file formats</b>: GDF (GUESS), GraphML (NodeXL), GML, NET (Pajek), GEXF and <a href="/users/supported-graph-formats/">more</a>.</p>
                <p class="tick"><b>Customizable by plugins</b>: layouts, metrics, data sources, manipulation tools, rendering presets and more.</p>
                <span class="learn_more"><a href="/features/">Learn More »</a></span>
