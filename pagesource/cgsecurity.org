<h2><span class="mw-headline" id="Data_recovery">Data recovery</span></h2>
<table>
<tr>
<td><a href="/wiki/TestDisk" title="TestDisk"><img alt="Testdisklogo clear 100.png" src="/mw/images/thumb/Testdisklogo_clear_100.png/50px-Testdisklogo_clear_100.png" width="50" height="50" srcset="/mw/images/thumb/Testdisklogo_clear_100.png/75px-Testdisklogo_clear_100.png 1.5x, /mw/images/Testdisklogo_clear_100.png 2x" /></a>
</td>
<td>If you have lost partition or strange problem with your hard disk partitions, run <a href="/wiki/TestDisk" title="TestDisk">TestDisk</a> to recover your data. TestDisk detects numerous filesystem including NTFS, FAT12, FAT16, FAT32, exFAT, ext2, ext3, ext4, btrfs, BeFS, CramFS, HFS, JFS, Linux Raid, Linux Swap, LVM, LVM2, NSS, ReiserFS, UFS, XFS.
<p>TestDisk can also undelete files from <a href="/wiki/TestDisk:_undelete_file_for_FAT" title="TestDisk: undelete file for FAT">FAT</a>, <a href="/wiki/Undelete_files_from_NTFS_with_TestDisk" title="Undelete files from NTFS with TestDisk">NTFS</a>, <a href="/wiki/TestDisk:_undelete_file_for_FAT" title="TestDisk: undelete file for FAT">exFAT</a> and <a href="/wiki/TestDisk:_undelete_file_for_ext2" title="TestDisk: undelete file for ext2">ext2</a> filesystem.
</p>
</td></tr>
<tr>
<td><a href="/wiki/PhotoRec" title="PhotoRec"><img alt="Photorec 64x64.png" src="/mw/images/thumb/Photorec_64x64.png/50px-Photorec_64x64.png" width="50" height="50" srcset="/mw/images/Photorec_64x64.png 1.5x" /></a>
</td>
<td>To recover your lost <a href="/wiki/Digital_Photos_Recovery_Using_PhotoRec" title="Digital Photos Recovery Using PhotoRec">digital pictures</a> or <a href="/wiki/File_Formats_Recovered_By_PhotoRec" title="File Formats Recovered By PhotoRec">lost files</a>, try <a href="/wiki/PhotoRec" title="PhotoRec">PhotoRec</a>. PhotoRec is a signature based file recovery utility. It handles more than 440 file formats including JPG, MSOffice, OpenOffice documents.
</td></tr></table>
