  <p><strong>WebRTC is a free, open project</strong> that provides browsers and mobile
applications with Real-Time Communications (RTC) capabilities via simple APIs.
The WebRTC components have been optimized to best serve this purpose.</p>

  <p><strong>Our mission:</strong> To enable rich, high-quality RTC applications to be
developed for the browser, mobile platforms, and IoT devices, and allow them
all to communicate via a common set of protocols.</p>

  <p>The WebRTC initiative is a project supported by Google, Mozilla and Opera,
amongst others. This page is maintained by the Google Chrome team.</p>

  <p><strong>NewÂ to WebRTC? Take a look at our <a href="https://codelabs.developers.google.com/codelabs/webrtc-web">codelab</a>.</strong></p>

  <p>Lots more resources for getting started are available from <a href="https://webrtc.org/start">webrtc.org/start</a>.</p>
